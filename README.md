# Ejecución

Para ejecutar el proyecto, se debe de usar maven y ejecutar el comando:

```
mvn clean jpro:run
```

Después de que que el comando se ejecute, la aplicacion va a abrir en el localhost en el puerto 8080 como aplicacion web.

```
http://localhost:8080
```
